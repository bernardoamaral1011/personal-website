  • Air Quality Visualisation Tool

    This tool, originally deployed in a digital ocean droplet, in an Apache server, fetches the air quality data in Lisbon from Qualar's website, runs an IDW interpolation on that data, and displays this data through extruded polygons with the Mapbox API in a webpage.

  • Web app for location-dependent message distribution

    Flask and MongoDB backend and jQuery frontend app, originally deployed in Google Cloud Platform. This app allows users, API admins and bots to send collective instant messages (RabbitMQ) to the app users depending on the user proximity to a target building (google maps API). The IST authentication protocol is implemented and users are required to have an IST account to log in.

  • Salary calculation tool for dentists

    This tool calculates an approximation of the net salary of a service provider hired on green receipts. It was made in vannilla js, bundled with parcel and deployed with netlify.

  • SQL Database for a hypothetical health care center

    Design of an SQL database. First, an E-R model was defined, followed by a Relational Model. Tables, triggers, queries, and functions were written. Finally, a website was created in HTML and PHP, which allowed clients to interact with the database.

  • Vigilance system with Raspberry Pi and Android Studio

    Raspberry Pi based video surveillance system with PTZ camera, microphone and WiFi connectivity. This project consists of 3 parts, the mobile app, coded in Android Studio, the Raspberry Pi code for camera control and IP update, and a small server built with PHP and SQL.

  • Culinary recipes mobile app

    Still under development. Ionic and Spring mobile application, which allows users to generate weekly meal plans through a very accessible UI and great UX.
